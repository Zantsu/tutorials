Grid'5000 Tutorials

API

curl-tutorial.sh
Learn how to use a simple command-line tool to interact with the API.
restfully-tutorial.rb
Learn how to use the Restfully library to abstract most of the repetitive work and use high-level objects.
resource-status.rb with 2.0 API or sid API
Learn how to discover the state of the resources with restfully
g5k-campaign-tutorial.rb
Learn how to use the g5k-campaign library to build reusable campaign engines.

For general information on the Grid'5000 API, please see https://www.grid5000.fr/mediawiki/index.php/API.

For links to developer documentation, head over to https://api.grid5000.fr.

The sources for the tutorials are available at http://github.com/grid5000/tutorials.
